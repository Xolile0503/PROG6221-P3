Cybersecurity Awareness Chatbot
Overview

The Cybersecurity Awareness Chatbot is a Windows Presentation Foundation (WPF) application. This application expands on the previous console-based chatbot by introducing a graphical user interface, database integration, task management, user personalisation, sentiment detection, and improved object-oriented programming practices.

The purpose of the chatbot is to educate users about cybersecurity awareness while providing an interactive and user-friendly experience.

Features
WPF Graphical User Interface
Modern WPF interface
User-friendly chat window
Interactive buttons and controls
Improved navigation and usability
Cybersecurity Chatbot

The chatbot can answer questions relating to:

Password security
Phishing attacks
Malware
Social engineering
Safe internet browsing
Multi-factor authentication
Cybersecurity best practices
Keyword Recognition

The chatbot analyses user input and recognises cybersecurity-related keywords to provide relevant responses.

MySQL Database Integration

The application stores and retrieves information using a MySQL database, including:

User profiles
Tasks
Activity logs
Quiz results
Task Manager

Users can:

Create cybersecurity reminder tasks
Edit tasks
Delete tasks
Mark tasks as completed
View all saved tasks
Reminder System

The chatbot reminds users about scheduled cybersecurity tasks and encourages good security habits.

User Personalisation

The chatbot remembers user information such as:

User name
Favourite cybersecurity topic
Previous interactions

This allows the chatbot to provide personalised responses.

Sentiment Detection

The chatbot performs simple sentiment analysis by identifying positive, negative, and neutral language to provide more appropriate responses.

Cybersecurity Quiz

The application includes an interactive quiz to test the user's cybersecurity knowledge.

Features include:

Multiple questions
Score calculation
Result storage
Activity Logging

The application records user activities, allowing interactions to be tracked and reviewed.

Error Handling

Exception handling has been implemented throughout the application to improve stability and prevent crashes.
